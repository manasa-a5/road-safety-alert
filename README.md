# Smart Vision System for Detecting Road Obstacles

##  Description
A deep learning-based system for real-time detection of road obstacles and accident prevention using YOLO.

##  Features
- Real-time obstacle detection
- Detects stationary objects
- Provides visual and audio alerts
- Works in low visibility conditions

## Technologies Used
- Python
- YOLO (You Only Look Once)
- OpenCV
- NumPy
- Flask

## System Requirements
- RAM: 8GB minimum
- Processor: Intel i5 or above
- Camera: Webcam/CCTV

## How to Run
1. Clone the repository
2. Install required libraries
3. Run the main Python file

## Output
Displays detected obstacles with alerts.
